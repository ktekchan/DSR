This file contains information about the various traces in this folder. 
These traces will be used to test the changes made in the DSR protocol and
assess the expected behavior made by these changes.

The traces are generated with the following requirements.
Degrees:
6  8  10 12 15
Mobility index based on max speed of the nodes:
2  5  10 15 20

The traces are named as follows:
deg_<int avg degree>_mob_<max speed of nodes>.tr
